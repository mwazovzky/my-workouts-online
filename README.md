## Start app

```
php artisan serve
```

## Linting

**Backend (PHP):**
```bash
./vendor/bin/pint
```

**Frontend (JavaScript/Vue):**
```bash
npm run lint        # Check for issues
npm run lint:fix    # Auto-fix issues
npm run format      # Format code with Prettier
```
